FINANCIERA NUEVO PROGRESO — Versión Moderna

Incluye:
- Tema oscuro profesional
- Dashboard con logo centrado
- Pantalla de bienvenida moderna con barra de carga
- Botones estilizados
- Ícono de app (assets/logo.ico)
- Script .bat para recompilar el .exe

Cómo ejecutar directamente (si tienes Python):
1) Abre una terminal en esta carpeta.
2) Ejecuta:  python main.py

Cómo generar el .exe portátil:
1) Doble clic en "crear_financiera_moderno.bat"
2) Al finalizar, encontrarás el ejecutable en: dist/Financiera.exe
3) También se crea "Financiera_Nuevo_Progreso.zip" con todo listo.
